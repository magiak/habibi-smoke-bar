# Habibi Smoke Bar

Website for **Habibi Smoke Bar** — a premium shisha lounge located in Ricany, Czech Republic.

## About

Habibi Smoke Bar offers premium hookah (shisha), signature cocktails, homemade lemonades, and a relaxed atmosphere. The venue also features PlayStation 4, darts, and quality tea & coffee.

- **Address:** Cernokostelecka 2361, 251 01 Ricany
- **Phone:** +420 604 237 508
- **Email:** habibismokebar@gmail.com
- **Instagram:** [@habibi_smoke_bar_ricany](https://www.instagram.com/habibi_smoke_bar_ricany/)

## Tech Stack

Single-page static website built with:

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (scroll effects, mobile menu, intersection observer)
- Google Fonts (Dancing Script, Montserrat, Playfair Display)

No build tools or dependencies required.

## Hosting

- Hosted on **GitHub Pages** from the `main` branch
- Repository: [magiak/habibi-smoke-bar](https://github.com/magiak/habibi-smoke-bar)
- Custom domain: **habibi.lukaskmoch.cz** (configured via `CNAME` file)

## Local Development

Open `index.html` directly in a browser — no server or build step needed.
